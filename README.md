# beecrowd
<p>beecrowd é uma plataforma digital de serviços profissionais. Capacitamos estudantes e profissionais de tecnologia e os conectamos às corporações 
mais inovadoras do mundo por meio de uma plataforma única, global e fácil de usar, para que serviços de TI e processos de recrutamento de alta qualidade 
possam ser fornecidos. Com uma base de usuários de mais de 600 mil desenvolvedores distribuídos em mais de 100 países (dados de 31 de dezembro de 2021), a 
beecrowd está entre as maiores comunidades de profissionais de tecnologia do mundo.</p>
